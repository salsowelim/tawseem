قواعد التقطيع

نستعرض فيما يلي الأجزاء القابلة للتقطيع

قبل الكلمة:

  • ( أ ) للاستفهام. مثال: أكِتابك؟
  • ( ب ) حرف جر. مثال: بكتابك.
  • ( س ) حرف استقبال. مثال: ستعلمون.
  • ( ف ) حرف عطف. مثال: خالد فـمحمد.
  • ( ك ) حرف جر للتشبيه. مثال: كالنار.
  • ( ل ) حرف جر قبل الاسم والفعل. مثال: لِمحمد.
  • ( و ) للعطف وللقسم. مثال: خالد و محمد.

بعد الكلمة:

  • ( ت ) تاء المتكلم. مثال: كتبتُ.
  • ( ت ) تاء المخاطب والمخاطبة. مثال: كتبتَ - كتبتِ.
  • ( ك ) كاف المخاطب والمخاطبة. مثال: ضربكَ - ضربكِ.
  • ( هـ ) هاء الغائب. مثال: كتابه.
  • ( ها ) هاء الغائبة. مثال: كتابها.
  • ( نا ) ناء المتكلمين. مثال: كتابنا.
  • ( تما ) للمثنى. مثال: أكلتما.
  • ( كما ) للتثنية. مثال: كتابكما.
  • ( ا ) ألف الغائبيْن. مثال: ضَرَبَا.
  • ( تا) تاء الغائبتيْن. مثال: ضربتا.
  • ( هما ) الغائب المثنى. مثال: ضربهما.
  • ( وا ) واو الجمع. مثال: ضربوا.
